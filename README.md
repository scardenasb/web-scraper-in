# Web scrapper to look for Linkedin python jobs in different countries.
>
<br></br>
### Description
> A Python3 web scrapper made withÂ [Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/bs4/doc/) includes a full integration with Postgresql, so every time the app is executed it sends the data parsed to the related sqlDB. Just need to create a ```database.ini``` file with this format:
```ini
[postgresql]
host=yourhost
user=yourdbuser
password=yourdbpassword
database=yourdbname
port=yourdbport
```
